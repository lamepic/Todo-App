# Todo-App

Awesome Todo-App made with react

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

```
Node
```

### Installing

```
Clone repo into directory
```

```
cd into directory
```

```
npm install
```

```
npm start
```
